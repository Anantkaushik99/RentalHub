# RentalHub 🏠

A full-stack rental marketplace for rooms, PGs, and flats.

## Features
- 🔍 Search and filter properties
- 🗺️ City-based search
- ⭐ Reviews and ratings
- 🔐 JWT Authentication
- 👤 Owner dashboard
- 📱 Responsive design

## Tech Stack
- Frontend: React, Vite, React Router
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Auth: JWT + bcrypt

## Setup
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
